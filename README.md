# 🏫 WEB ISM INCIDENCIAS
[![Descargar Última Versión](https://img.shields.io/badge/Descargar-Última_Versión-blue?style=for-the-badge&logo=github)](https://github.com/AlejandroGonzaloBarroso/WEB_ISM_INCIDENCIAS/releases/latest)
Un ecosistema descentralizado en tiempo real para la gestión, notificación y control integral de incidencias técnicas y de mantenimiento. 
El proyecto consta de una infraestructura cruzada (*Cross-Platform*) impulsada por **Firebase**, que permite a los usuarios reportar problemas desde su dispositivo móvil, mientras que los administradores reciben, gestionan y chatean sobre dichas incidencias en tiempo real desde un panel de control corporativo para PC/Mac.
Cuenta con un **motor de actualizaciones automáticas (OTA)** integrado: todas las aplicaciones (escritorio y Android) parchean sus errores e instalan las nuevas versiones por sí mismas de forma completamente silenciosa y sin intervención del usuario.

## 💻 Requisitos del Sistema
### 📱 Android (`.apk`)
- **Mínimo:** Android 7.0 (Nougat) / API Nivel 24
- **Recomendado:** Android 11.0 o superior
- **Memoria RAM:** 2 GB mínimo / 4 GB recomendado
- **Almacenamiento:** ~50 MB de espacio libre
- *Nota:* Requiere que la aplicación del sistema "Android System WebView" esté actualizada.
### 🪟 Windows (`.exe` Installer)
- **Mínimo:** Windows 10 (Arquitectura de 64 bits)
- **Recomendado:** Windows 10 o Windows 11
- **Procesador:** CPU de doble núcleo a 2 GHz
- **Memoria RAM:** 4 GB mínimo / 8 GB recomendado
- **Almacenamiento:** ~200 MB de espacio libre
- *Nota:* Versiones obsoletas como Windows 7 y 8.1 no están soportadas oficialmente por el motor por políticas de seguridad.
### 🍏 macOS (`.dmg`)
- **Mínimo:** macOS 10.15 (Catalina)
- **Recomendado:** macOS 12 (Monterey) o superior
- **Arquitectura:** Totalmente nativo tanto para Intel (x64) como para Apple Silicon (M1/M2/M3/M4).
- **Memoria RAM:** 4 GB mínimo / 8 GB recomendado
- **Almacenamiento:** ~200 MB de espacio libre
### ⚠️ iOS / iPadOS (`.ipa`)
> [!WARNING]  
> **Uso Restringido:** El archivo iOS descargable desde las *Releases* es un archivo `.ipa` **sin firmar** (Unsigned). Debido a las estrictas políticas del ecosistema cerrado de Apple, esta versión **solo se puede instalar si tienes el iPhone en "Modo Desarrollador"** y conectas el dispositivo al ordenador utilizando herramientas externas como *AltStore* o *Sideloadly*. 
> *Para una instalación normal de cara a usuarios estándar, es imprescindible publicar la aplicación oficialmente a través de la App Store de Apple.*
